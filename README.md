# OSINT Tool V1

An open source OSINT automation tool for security researchers.
Built by zeo-design.

## What It Does
Automates manual reconnaissance tasks that normally take 20+ minutes.

## Modules
- Domain Recon — WHOIS, DNS, SSL, SPF, DKIM
- Email Intelligence — Email format guessing
- Google Dorking — Auto generates 10 dork queries with clickable links

## Requirements
- Python 3.11+
- Git

## Installation
git clone https://github.com/zeo-design/osint-tool.git
cd osint-tool
pip install -r requirements.txt
## Usage
python main.py
Select a module from the menu and enter your target domain.
