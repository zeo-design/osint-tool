Auto OSINT Tool

An open source AI-assisted reconnaissance and OSINT framework built for cybersecurity learning, infrastructure visibility, and security research.

About The Project

This project was created to help security researchers, students, analysts, and builders understand how reconnaissance and intelligence gathering work in real-world environments.

Instead of only using existing tools, we wanted to learn by building our own modular reconnaissance framework from the ground up.

The goal is to automate repetitive reconnaissance tasks while keeping the framework clean, extensible, and educational.

This project is currently under active development.

---

Current Features

Phase 1 — Core Reconnaissance

- WHOIS intelligence
- DNS enumeration
- SSL certificate analysis
- SPF / DKIM inspection
- Email pattern generation
- Automated Google dorking

Phase 2 — Infrastructure Intelligence (In Progress)

- Subdomain enumeration
- IP geolocation and ASN lookup
- Infrastructure visibility
- Multi-IP reconnaissance workflows
